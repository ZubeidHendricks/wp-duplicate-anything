One-click duplicate any post, page or custom post type — including taxonomies and custom fields.

== Description ==

Duplicate Anything adds a **Duplicate** link to every post, page and custom
post type row. Click it and you get an exact copy as a new draft — title,
content, excerpt, categories, tags, and custom fields all carried over.

No setup required. No dashboard ads.

**Free features**

* Duplicate any post, page or custom post type.
* Copies taxonomies (categories, tags, custom) and post meta.
* Choose: open the copy in the editor, or return to the list.
* Choose: always draft, or match the original status.

**Pro features**

* Bulk duplicate many items at once.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Hover any post/page row and click **Duplicate**.

== Frequently Asked Questions ==

= Does it copy custom fields? =

Yes — all post meta except internal edit locks is copied.

= Does it work with custom post types? =

Yes, any public post type with a row-action list.
